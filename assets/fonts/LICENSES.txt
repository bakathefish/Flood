How the static instances were produced
--------------------------------------
The IBM Plex Sans and Bricolage Grotesque files are single-weight static
instances cut from the official Google Fonts variable sources using
fontTools.varLib.instancer (all axes pinned to default except wght, which is
pinned to the target weight; the name table was normalised so each family
reports a single family name with the weight in the subfamily / usWeightClass).
This keeps per-weight selection reliable in matplotlib. IBM Plex Mono was small
enough to ship the upstream static files directly.
